Introduction

『HACHI Membership Checker』是 HoneyBEES Club (HACHI 中文群) 專用的 Chrome 瀏覽器插件。

主要功能為「提供簡單能在Discord領取會員身分組」的方法，並作為其他工具的入口。

如若使用上有任何問題，請至 Discord Server 透過 #聯絡我們 詢問。
